Vue-C3 Charts
===============

Vue-c3 is a node module that makes working with the c3 graphs library a little simpler and allowing you to treat these charts are vue-like components.

## Installation

```
// Using npm
npm install vue-c3
// Using Yarn
yarn add vue-c3
```
